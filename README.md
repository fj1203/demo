## 方证云用户中心

## Install
```bash
npm install
```
## Run
### Development
```bash
npm run dev
```
### Development Proxy
```bash
npm run dev --proxydev
```
### Production(Test)
```bash
npm run deploy
```
### Production(Build)
```bash
npm run build
```
